# 页面说明
### 布局 —— Grid
 * 页面须以栅格化布局为准则
 * 页面子元素对齐方向以下为准:
    * 子元素向左排列:`justify="start"`
    * 子元素向右排列:`justify="end"`
    * 子元素居中排列:`justify="center"`
    * 子元素等宽排列:`justify="space-between"`
    * 子元素分散排列:`justify="space-around"`

如例：

```
<Row type="flex" justify="start">
  <Col span="12">内容</Col>
  <Col span="12">内容</Col>
</Row>
```
